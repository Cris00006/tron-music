 # TRON MUSIC

TRON MUSIC is an experimental music streaming platform developed as part of the **Tron Labs** ecosystem.

The project explores new models for music distribution, monetization, and digital consumption, combining a modern backend a cross-platform fronted, and scalable system architecture

## Tech Stack

**Backend**
- Python
- FastAPI
-SQLAlchemy

**Fronted**
-Flutter (cross-platform)

**Infrastructure**
- Git + GitHub
- Modular, scalable architecture

## Project Status
> Active development
This repository currently contains the **base structure and core foundations** of the TRON MUSIC platform
